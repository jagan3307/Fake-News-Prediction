📰 Fake News Detection + Chatbot

Fake News Detection + Chatbot is an advanced Machine Learning and Natural Language Processing (NLP) based web application designed to identify whether a news article or headline is Fake or Real. The project uses supervised learning techniques to analyze textual content, preprocess news data, extract meaningful features using TF-IDF vectorization, and classify the news using a trained machine learning model.

The application also includes an interactive chatbot that allows users to ask questions related to fake news detection, machine learning, and NLP concepts. A user-friendly Streamlit interface is integrated to provide real-time predictions, chatbot interaction, and data visualization dashboards.

This project demonstrates the complete workflow of an end-to-end Machine Learning application, including data preprocessing, model training, prediction, model deployment, and web application development.

🎯 Project Objective

The main objective of this project is to reduce the spread of misinformation by automatically identifying suspicious or fake news content using Machine Learning techniques and providing users with an easy-to-use interactive platform.
